# Security Policy

## Supported Repositories

This organization-level policy applies to repositories under:

- https://github.com/moaiy-com

If a repository contains its own `SECURITY.md`, that repository-specific policy
takes precedence.

## Reporting a Vulnerability

Please do **not** report vulnerabilities in public issues.

Use one of these private channels:

- Email: `security@moaiy.com`
- GitHub private vulnerability reporting (if enabled in the target repository)

Include:

- Affected repository and version/commit
- Reproduction steps or proof of concept
- Impact and suggested severity
- Any mitigation ideas (optional)

## Response Targets

- Initial acknowledgement: within 72 hours
- Triage update: within 7 business days
- Fix timeline: based on severity and exploitability

## Safe Harbor

We appreciate responsible disclosure and good-faith research. Please avoid
privacy violations, service disruption, and data destruction.
